Installation of 2404 in desktop
--------------------------
1. Download Ubuntu 24 LTS from website. Do it in Apple M3 Pro. "Etcher for macOS (arm64)".
2. Download Balena Etcher in Mac.
3. Plug in formatted USB
4. Open Etcher, select USB, select downloaded ubuntu iso and flash it.
5. Remove the USB and plug it in desktop.
6. Start the desktop and keep pressing F2/F12/F10 to enter Boot menu.
7. Select bootmenu and in boot menu select to install Ubuntu from USB.Slect - UEFI: SanDisk
8. Complete installation as guided in installation UI

install n8n using docker
------------------------
> sudo apt update && sudo apt upgrade -y
> sudo apt-get install -y ca-certificates curl gnupg

# Add Docker’s official GPG key
> sudo install -m 0755 -d /etc/apt/keyrings
> curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
> sudo chmod a+r /etc/apt/keyrings/docker.gpg

# Add repository
> echo \
    "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] \
    https://download.docker.com/linux/ubuntu \
    $(lsb_release -cs) stable" | \
    sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
#  Install Docker + Compose plugin
> sudo apt update
> sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
> docker compose version

#  Install n8n in docker
> mkdir -p ~/n8n
> cd ~/n8n
> nano docker-compose.yml

version: '3.8'

services:
  n8n:
    image: n8nio/n8n:latest
    restart: always
    ports:
      - "5678:5678"
    volumes:
      - ./n8n_data:/home/node/.n8n
    environment:
      - GENERIC_TIMEZONE=Asia/Kolkata
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=admin

> mkdir -p ~/n8n/n8n_data
> sudo chown -R 1000:1000 ~/n8n/n8n_data
> sudo groupadd docker
> sudo usermod -aG docker $USER
> newgrp docker
> docker compose up -d
> docker ps

Integrate slack with n8n
------------------------
1. Create free slack account with your google mail.
2. Go to apps website to create slack app. https://api.slack.com/apps/A09DMKKJYSC
3. Give name as n8n Integration
4. Click on OAuth & Permissions
5. In User Token Scopes add "admin" scope
6. Go to Scopes. Add below scopes - 
    app_mentions:read
    channels:read
    chat:write
    conversations.connect:read
    conversations.connect:write
    groups:read
    im:read
    mpim:read
    users:read
7. In the "Basic Information" tab, in "Display Information" section give short description and Background color.
8. Save Changes
9. Click on Install 
10. Click on OAuth & Permissions -> OAuth Tokens -> Install to AnveshanWS - xoxb- token
11. After installation, you’ll see a "Bot User OAuth Token" That you need to use in n8n
    a. In n8n, add a Slack node.
    b. Under Authentication, choose Access Token.
    c. Paste your xoxb- token
12. In the Basic Information, you have Signing Secret, use that in n8n. n8n asks for Access Token and Signing Secret.
